    Move an element in a sequence from one index to another.

    Type parameters

    • T

    Parameters

    • object: MutableSequenceOrArrayLike

      The sequence or array-like object to mutate.

    • fromIndex: number

      The index of the element to move.

    • toIndex: number

      The target index of the element.

      Complexity

      Linear.

      Undefined Behavior

      A fromIndex which is non-integral or out of range.

      A toIndex which is non-integral or out of range.

      Example

      import { move } from 'phosphor/lib/algorithm/mutation';

let data = [0, 1, 2, 3, 4];
move(data, 1, 2);  // [0, 2, 1, 3, 4]
move(data, 4, 2);  // [0, 2, 4, 1, 3]

reverse

  • reverse<T>(object: MutableSequenceOrArrayLike, first?: number, last?: number): void
    • Defined in algorithm/mutation.ts:84

    Reverse a sequence in-place subject to an optional range.

    Type parameters

    • T

    Parameters

    • object: MutableSequenceOrArrayLike

      The sequence or array-like object to mutate.

    • Optional first: number

      The index of the first element of the range. This should be <= the last index. The default is 0.

    • Optional last: number

      The index of the last element of the range. This should be >= the first index. The default is length - 1.

      Complexity

      Linear.

      Undefined Behavior

      A first index which is non-integral or out of range.

      A last index which is non-integral or out of range.

      Example

      import { reverse } from 'phosphor/lib/algorithm/mutation';

let data = [0, 1, 2, 3, 4];
reverse(data, 1, 3);  // [0, 3, 2, 1, 4]
reverse(data, 3);     // [0, 3, 2, 4, 1]
reverse(data);        // [1, 4, 2, 3, 0]

    Returns void

rotate

  • rotate<T>(object: MutableSequenceOrArrayLike, delta: number): void
    • Defined in algorithm/mutation.ts:135

    Rotate the elements of a sequence by a positive or negative amount.

    Type parameters

    • T

    Parameters

    • object: MutableSequenceOrArrayLike

      The sequence or array-like object to mutate.

    • delta: number

      The amount of rotation to apply to the elements. A positive value will rotate the elements to the left. A negative value will rotate the elements to the right.

      Complexity

      Linear.

      Undefined Behavior

      A delta amount which is non-integral.

      Example

      import { rotate } from 'phosphor/lib/algorithm/mutation';

let data = [0, 1, 2, 3, 4];
rotate(data, 2);   // [2, 3, 4, 0, 1]
rotate(data, -2);  // [0, 1, 2, 3, 4]
rotate(data, 10);  // [0, 1, 2, 3, 4]
rotate(data, 9);   // [4, 0, 1, 2, 3]

    Returns void
