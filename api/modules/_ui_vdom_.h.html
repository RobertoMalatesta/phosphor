    • Defined in ui/vdom.ts:667

    Create a new virtual DOM node.

    Parameters

    • tag: "a"

      The tag name for the node.

    • Optional attrs: IAnchorAttrs

      The attributes for the node, if any.

    • Rest ...children: FactoryChild[]

      The children for the node, if any.

    Returns VNode

    A new virtual DOM node for the given parameters.

    Notes

    The children may be string literals, other virtual DOM nodes, or an array of either of those things. String literals are converted into text nodes, and arrays are inlined as if their contents were given as positional arguments. This makes it easy to build up an array of children by any desired means. null child values are ignored.

    A strongly typed bound function for each tag name is also available as a static attached to the h() function. E.g. h('div', ...) is equivalent to h.div(...).
